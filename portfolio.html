Java Script Snake Game

This is the snake game that I made, this game uses JavaScript to operate. The point of the game is to try to get the highest score as possible. The score is displayed in the canvas where the game is played. To play the game you use the arrow keys on the keyboard. Eat food to grow longer and make the score higher, but don't run into the wall or the snake or you die!
